# A Grande Expedição do Ícaro

> **Um convite digital inspirado na imaginação de Júlio Verne, nos atlas ilustrados e na delicadeza dos livros infantis.**

## Sobre o projeto

**A Grande Expedição do Ícaro** é um convite online desenvolvido para celebrar o primeiro aniversário de Ícaro.

Mais do que uma página informativa, o projeto foi concebido como uma experiência narrativa, conduzindo o visitante por pequenos capítulos de uma grande aventura. A proposta combina literatura, exploração, infância e artesanato para criar uma identidade visual afetiva, elegante e atemporal.

A direção artística foi inspirada em livros ilustrados europeus, mapas de exploração, dioramas artesanais e cenografias construídas em feltro, reinterpretando o espírito aventureiro das obras de Júlio Verne para um universo infantil.

## Conceito

O convite apresenta a festa como um livro ilustrado, no qual cada seção representa um novo capítulo da jornada de Ícaro.

A narrativa é conduzida pela ideia de que:

> **"Neste primeiro ano, Ícaro iniciou sua maior aventura: descobrir o mundo. Entre os céus e os oceanos, cada nova descoberta tornou-se uma lembrança inesquecível."**

Esse conceito orienta toda a arquitetura visual, tipografia, paleta cromática e organização dos conteúdos.

## Objetivos

* transformar um convite em uma experiência digital;
* comunicar afeto, descoberta e imaginação;
* utilizar uma linguagem visual minimalista e editorial;
* privilegiar simplicidade e espaços negativos;
* criar uma identidade visual consistente para toda a festa;
* servir como base para futuras peças gráficas e digitais.

## Direção de Arte

A identidade visual foi construída a partir de referências como:

* literatura de aventuras do século XIX;
* imaginação científica de Júlio Verne;
* atlas ilustrados infantis;
* livros sensoriais em feltro (*quiet books*);
* mapas de exploração;
* dioramas artesanais;
* cenografia infantil contemporânea;
* ilustrações editoriais minimalistas.

## Paleta Cromática

A paleta procura transmitir leveza, acolhimento e natureza.

* Azul Céu
* Azul Oceano
* Verde Sálvia
* Marfim
* Bege Areia
* Terracota
* Madeira Natural
* Cobre Envelhecido

## Tipografia

O projeto utiliza uma combinação entre elegância clássica e excelente legibilidade:

* **Cormorant Garamond** — títulos e elementos editoriais;
* **Nunito** — textos e interface.

## Organização Narrativa

O site é estruturado como um pequeno livro.

* Hero
* Capítulo I — Toda grande aventura começa com um primeiro passo
* Capítulo II — Um mundo inteiro espera por um pequeno explorador
* Capítulo III — Nosso próximo destino
* Capítulo IV — Informações da Expedição
* Capítulo V — Confirme sua presença
* Epílogo

## Princípios de Design

Durante o desenvolvimento foram adotados os seguintes princípios:

* HTML5 semântico;
* CSS organizado em um Design System simples;
* layout responsivo;
* arquitetura preparada para evolução futura;
* identidade visual consistente;
* narrativa contínua;
* acessibilidade;
* simplicidade visual;
* hierarquia tipográfica clara;
* componentes reutilizáveis.

## Evoluções Planejadas

O projeto foi concebido para permitir futuras melhorias, entre elas:

* ilustrações vetoriais próprias em SVG;
* animações suaves com JavaScript;
* texturas de feltro aplicadas aos elementos gráficos;
* RSVP integrado;
* mapa interativo;
* contagem regressiva;
* trilha sonora opcional;
* modo de impressão;
* otimizações de desempenho e SEO.
